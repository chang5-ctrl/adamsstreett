# Contributing

Thank you for your interest in contributing to Adams Streett Partners.

This repository represents proprietary financial infrastructure. Contributions are restricted to authorized collaborators.

## Development Guidelines

- Maintain clean and readable code
- Follow existing architecture patterns
- Document any new modules or integrations
- Ensure security best practices are followed

## Pull Requests

All pull requests must:

- Clearly describe the proposed change
- Include appropriate documentation
- Pass internal testing requirements

Unauthorized forks or derivative financial platforms based on this repository are not permitted.
