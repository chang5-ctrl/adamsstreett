# Security Policy

Adams Streett Partners takes the security of its platform and infrastructure seriously.

## Supported Versions

Security updates and patches are applied to the latest production release of the platform.

## Reporting a Vulnerability

If you discover a security vulnerability within the platform, please report it responsibly.

Submit reports to:

security@adamsstreettpartners.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested remediation if available

All valid security reports will be reviewed and addressed as quickly as possible.
